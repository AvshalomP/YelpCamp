RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index / GET List all Comment.find()
New /new GET Show new campground form N/A
Create / POST Create a new campground, then redirect somewhere Comment.create()
Show /:comment_id GET Show info about one specific campground Comment.findById()
Edit /:comment_id/edit GET Show edit form for one campground Comment.findById()
Update /:comment_id PUT Update particular campground, then redirect somewhere Comment.findByIdAndUpdate()
Destroy /:comment_id DELETE Delete a particular campground, then redirect somewhere Comment.findByIdAndRemove()
